# Distribution Downloader

Ce dossier contient les fichiers a publier sur un hebergement gratuit.

## Fichiers

- `index.html`: page de telechargement.
- `vanta-tv.apk`: APK signe a telecharger.
- `privacy.html`: politique de confidentialite.
- `terms.html`: conditions d'utilisation.
- `version.json`: metadata simple.

## Publication gratuite

Options simples:

- GitHub Pages
- Cloudflare Pages
- Netlify

Pour Downloader, utiliser une URL courte qui pointe vers `index.html`.
